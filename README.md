# AI 뉴스 브리핑 웹앱

## 1. 프로젝트 소개

이 프로젝트는 매일 AI 관련 뉴스를 수집하고, 생성형 AI를 활용해 중요한 뉴스 TOP 6를 선정한 뒤 카드뉴스 형태로 보여주는 Streamlit 기반 웹앱입니다.

AI 뉴스는 빠르게 변화하고 정보량도 많기 때문에 일반 사용자가 매일 중요한 흐름을 파악하기 어렵습니다. 이 문제를 해결하기 위해 RSS로 AI 뉴스를 수집하고, Gemini API를 활용해 주요 뉴스를 선별·요약하여 사용자가 한눈에 볼 수 있는 브리핑 페이지를 제작했습니다.

## 2. 프로젝트 목적

본 프로젝트의 목적은 다음과 같습니다.

- 매일 AI 관련 주요 뉴스를 빠르게 확인할 수 있는 웹앱 제작
- 생성형 AI를 활용한 뉴스 선별 및 요약 자동화
- 뉴스 데이터를 날짜별 JSON 파일로 저장하여 재조회 가능하게 구성
- 카드뉴스 형태의 UI를 통해 정보 접근성 향상
- 생성형 AI 활용 과정에서 정확성, 저작권, 편향성, 개인정보 보호 문제를 함께 고려

## 3. 주요 기능

### 1) AI 뉴스 RSS 수집

AI타임즈 RSS 데이터를 활용하여 최신 AI 뉴스 목록을 수집합니다.

### 2) Gemini API를 활용한 TOP 6 뉴스 선정

수집한 뉴스 목록을 Gemini API에 전달하여 중요도, 최신성, 대중성, 다양성, 교육적 가치를 기준으로 오늘의 AI 뉴스 TOP 6를 선정합니다.

### 3) 날짜별 JSON 저장

선정된 뉴스 데이터는 날짜별 JSON 파일로 저장됩니다.

├─ 2026-06-05.json
